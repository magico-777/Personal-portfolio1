## 🛠 Technology Stack
This project was developed using a clean and efficient frontend architecture:

* **HTML5 & CSS3:** Semantic structure and custom styling (Glassmorphism).
* **Bootstrap 5.3:** For the responsive 12-column grid system and core UI components.
* **Vanilla JavaScript (ES6+):** For logic, theme switching, and interactive elements.
* **EmailJS API:** A serverless solution for real-time contact form submissions.
* **Git & GitHub:** For version control and seamless deployment.

---

## ✨ Key Features
* 🌓 **Smart Dark/Light Mode:** Dual-theme support with persistence via `localStorage`.
* 📱 **Fully Responsive:** Mobile-first design, optimized for all screen sizes.
* 💎 **Glassmorphism UI:** Modern frosted-glass effects with neon blue accents.
* 📧 **Direct Messaging:** Functional contact form that sends messages directly to my email.
* ⚡ **Performance:** Lightweight design with minimal external dependencies.

## ⚙️ Setup & Installation

To run this project locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/magico-777/Personal-portfolio1.git](https://github.com/magico-777/Personal-portfolio1.git)
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd Personal-portfolio1
    ```
3.  **Open the project:**
    * Simply double-click `index.html` to open it in your browser.
    * *Recommended:* Use the **Live Server** extension in VS Code.
